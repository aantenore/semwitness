# Security policy

Please do not open public issues for suspected vulnerabilities or prompt/content leaks. Use GitHub's private vulnerability reporting for this repository.

SemWitness is experimental and defaults to shadow mode. Treat every codec result as untrusted until its proof has been verified against the same pinned policy, codec registry, tokenizer fingerprint, and local content store.

Supported security fixes target the latest `0.x` release line. Reports should include the SemWitness version, operating system, minimal reproduction, expected invariant, and whether original content may have left the local machine. Do not include live secrets or private prompts.
